 Scientific Calculator

A powerful Flutter-based calculator application built for students, engineers, and professionals who need both standard and advanced mathematical tools in a single interface.

Features

* Basic arithmetic operations including addition, subtraction, multiplication, and division
* Trigonometric calculations: sin, cos, tan, asin, acos, and atan
* Hyperbolic functions: sinh, cosh, and tanh
* Logarithmic operations: log (base 10) and natural log (ln)
* Powers and roots: square, cube, square root, and reciprocal values
* Advanced mathematical operations such as factorial, permutations (nPr), and combinations (nCr)
* Quick access to mathematical constants π and e
* Toggle between Degree and Radian angle modes
* Percentage calculations and positive/negative sign switching
* Backspace and clear functions for easy editing

State Management

The application uses Riverpod to manage state in a scalable and maintainable way.

Getting Started

flutter pub get
flutter run

Requirements

* Flutter 3.x or later
* Dart 3.x or later
